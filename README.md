## :memo: Description 

```sh
하늘에 별도 따다주고 싶은 내 여자친구, 데이트 코스 짜느라 두통, 치통, 마음의 고통까지 겪고 계시나요?
더 이상 고민은 Stop! 당신의 사랑에도,'라인'이 필요합니다.
LoveLine은 빅데이터 분석 기반 데이트 코스 추천 웹 어플리케이션입니다.
```
